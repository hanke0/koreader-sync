This is a Go implementation of the [koreader-sync-server](https://github.com/koreader/koreader-sync-server), using SQLite as the backend storage for convenient self-hosting.

# How to use.
Download the binary at [release page](https://github.com/hanke0/koreader-sync/releases) and setup by command:
```
koreader-sync -addr 127.0.0.1:9200 -db koreader.sqlite
```
